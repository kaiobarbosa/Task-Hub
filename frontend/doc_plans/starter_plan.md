# Site de tasks

    - Um site que o usuário poderá criar, editar, excluir e atualizar uma task.
        - As tasks sao relacionadas ao usuario
    - Cada usuario deve criar sua própria conta para poder utilizar o site.
    - Na tela inicial, a Index Screen, o usuario irá ver um resumo do site
        - Prints das principais funcionalidades
        - Resumos acompanhando os prints
        - No começo da pagina deve conter um botao para "acessar o conteudo"
            - O botao levará ao fim da pagina com a opcao de se cadastrar ou fazer o login. 
        - ainda na tela inicial deverá ter um rodapé com alguma informacoes
            - Linkedin
            - Instagram
            - Resumo da aplicaçao 

    - Telas apos o login

        - Home
            - Resumo das tasks mais recentes criadas
            - Uma "tabela" ou "calendario" para marcar os dias que de fato ouve uma conclusa de uma tarefa
                - Posteriormente da para criar um sistema de "Ofensiva" como o do Duolingo e B7
            - Opcao para criar um grafico com a frequencia ou quantidade de task concluidas em determinado espaço de tempo
            
        - Perfil
            - Edicao das proprias infomacoes
                - Clissificacao da importancia das informacoes, dependendo da importancia devera ser necessario "confirmacoes" para a modificacao.
                    -  baixa - sem confirmacao alem de um segundo clique para ter certeza
                        - Nome
                        - Sobrenome
                        - Idade
                    - Alta - confirmacao com a senha
                        - Email
                        - Número de telefone
                    - Altissima - confirmacao via email
                        - Senha
        - Crud de tarefas

            - Criar tarefas
            - Editar
            - Atualizar
            - Deletar

            - Tudo na mesma tela, apenas com botoe para cada funcao
                - Deletar deve ter confirmacao de certeza

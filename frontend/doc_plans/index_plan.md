## INDEX.HTML

# Resumo do index.html
- A pagina index, vai ser o porta de entrada para o site. De inicio o usuario ira ver o nome do site: "TASK-HUB", logo a baixo um botao para acessar a ferramenta. Abaixo do botao o usuario ira ver um resumo das principais funcionalidades da aplicacao:

    item 1 - Criacao de tasks
    item 2 - Acompanhamento da 'Ofensiva', sequencia diaria de conclusao de tasks
    item 3 - Gerenciamento das tasks ja concluidas e pendentes
    item 4 - Criacao de graficos baseados na quantidade de tasks concluidadas em determinado espaco de tempo.

    Esse resumo devera conter no maximo dois paragrafos por item. 
    Cada item vem acompanhado da sua respectiv imagem ilustrando o mesmo(item 1 - imagem 1 // item 2 - imagem 2 // item 3 - imagem 3 // item 4 - imagem 4)

    Abaixo dos 4 resumos, o usuario ira ver um botao para logar no sistema. Ao clicar o usuario sera mandado para a tela de login.

    No final da pagina devera conter um rodape com as seguintes informacoes:
        - github - @kaiobarbosa
        - linkedin - @kaiobarbosa

        - Um breve resumo sobre a funcao em si do site:
            O site foi desenvolvido com a finalidade de auxiliar aqueles que tem dificuldade em terminar tarefas diarias seja por esquecer, se destrair, desistir,  etc. Por meio da 'Ofensiva' o usuario é incentivado a completar todos os dias tarefas criadas por ele mesmo, alem de poder gerenciar e organizar os seus afazeres do dia/semana/mes.

# Scopo do projeto

    -  todo a pagina em questao devera seguir rigorosamente estas regras chaves de css.
        - o site devera ser em tema dark, escuro, por padrao
        - pode ser adicionado detalhes roxo pela pagina.
        - a cor da fonte dos p, h1, h2, e fontes dos botoes devera ser lilas claro

# Header Section

- no topo da pagina. deverá ter uma navbar com os seguintes elementos:
    - Logo da págna á esquerda
        -  A logo devera ser feita com html e css "Th</>", exatamente assim
        -  Logo lilas com sombra roxa em volta
    - Botao de funcinalidades, sobre e acesse ja
        - os tres botoes a direita, seguindo essa ordem da esqueda para a direita
        - Cada botao devera redirecionar para o trecho especifico dentro da propria pagina
            - funcionalidades - manda o usuario para o trecho dos prints
            - sobre - manda o usuario para o rodapé no final do site.
            - acesse já - manda o usuario para o trecho onde esta o botao de login - apos os prints e antes do rodapé
